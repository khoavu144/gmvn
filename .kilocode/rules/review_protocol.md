# AI Code Review Protocol

All AI-generated code must be reviewed.

## Review Checklist

Architecture
- Does it respect architecture rules?

Security
- Any secrets exposed?

Performance
- Any inefficient loops or queries?

Readability
- Is the code understandable?

Diff Size
- Is the change minimal?

## Approval Levels

Small change:
1 reviewer

Architecture change:
2 reviewers

Major system change:
team discussion required
