# Architecture Map

## System Layers

Client
↓
API Layer
↓
Service Layer
↓
Domain Logic
↓
Database

## Responsibilities

Client:
UI logic and user interactions.

API Layer:
Handles requests and responses.

Service Layer:
Business logic orchestration.

Domain:
Core domain rules.

Database:
Persistent storage.

## Principles

- Separation of concerns
- Stateless services
- Predictable APIs
