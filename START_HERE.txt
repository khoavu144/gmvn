🔌 API (43 ENDPOINTS)

Authentication (6):
  POST /auth/register           - Create account
  POST /auth/login              - Login
  POST /auth/refresh            - Refresh token
  POST /auth/logout             - Logout
  POST /auth/forgot-password    - Reset request
  POST /auth/reset-password     - Reset confirm

User Management (5):
  GET /users/me                 - Current user
  PUT /users/me                 - Update profile
  PUT /users/me/avatar          - Upload photo
  GET /users/me/progress        - Get progress history
  POST /users/me/progress       - Log weight/measurements

Trainer Discovery (4):
  GET /trainers                 - List all trainers
  GET /trainers/:id             - Get trainer profile
  GET /trainers/:id/programs    - Get programs
  GET /trainers/:id/reviews     - Get reviews

Trainer Dashboard (7):
  GET /trainers/dashboard/overview      - Stats
  POST /trainers/programs               - Create program
  PUT /trainers/programs/:id            - Edit program
  POST /trainers/programs/:id/publish   - Publish
  GET /trainers/dashboard/clients       - Client list
  GET /trainers/dashboard/clients/:id   - Client detail
  GET /trainers/dashboard/payments      - Revenue

Subscriptions (5):
  POST /subscriptions                   - Subscribe
  POST /subscriptions/confirm-payment   - Confirm Stripe
  GET /subscriptions/me                 - My subscriptions
  GET /subscriptions/:id/workouts       - Get workouts
  POST /workouts/:id/log                - Log workout

Messaging (3):
  GET /messages/conversations           - Conversations
  GET /messages/conversations/:id       - Messages
  POST /messages/send                   - Send message

Reviews (3):
  POST /reviews                         - Create review
  PUT /reviews/:id                      - Edit review
  GET /trainers/:id/reviews             - Get reviews

WebSocket (Real-time):
  message:send                  - Send via Socket
  message:receive               - Receive via Socket
