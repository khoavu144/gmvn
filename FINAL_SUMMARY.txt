🔌 NEW API ENDPOINTS (25 new endpoints)

Public Endpoints:
  GET /profiles/:slug                 → Get trainer profile by URL slug
  GET /trainers/:id/profile           → Get trainer profile by ID
  GET /trainers/:id/profile/export/pdf → Download as PDF
  GET /trainers/:id/profile/export/json → Export as JSON
  POST /profiles/:slug/view           → Track profile view

Trainer Endpoints:
  PUT /trainers/me/profile            → Update profile
  PUT /trainers/me/profile/customization → Change theme/layout
  POST /trainers/me/profile/gallery   → Upload gallery images
  POST /trainers/me/profile/experience → Add work experience
  POST /trainers/me/profile/certifications → Add certifications
  GET /trainers/me/profile/analytics  → View profile analytics

Admin Endpoints:
  GET /admin/profile-fields           → List all available fields
  PUT /admin/profile-fields/:id       → Update field config
  POST /admin/profile-fields          → Create custom field
  DELETE /admin/profile-fields/:id    → Delete field
  + More for templates & global settings
